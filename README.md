Messaging with Faye
---------


This is a quick build of a full Ruby on Rails 3.2 application implementing a simple chat application using Faye for the St. George, UT Ruby Meetup group.


Gems being used
---------
    gem 'rails', '3.2.3'

    gem 'mongoid', '~> 3.0.0rc'
    gem 'moped', '~> 1.0.0.rc'

    gem 'thin'
    gem 'private_pub'

    gem 'jquery-rails'
    gem 'haml-rails'
